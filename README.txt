# Cat Rooftop Adventure

Greetings to all developers and judges of js13kGame! My name is Vladislav, and this is my first game created specifically for this competition. I'm very excited to participate and share my creation with you!

## About the Game

In "Cat Rooftop Adventure," you need to help the cat climb as high as possible across the rooftops!

## Controls

•   **[Spacebar]**: Jump
•   **[Right Arrow Key]**: Move Right
•   **[Left Arrow Key**]: Move Left

## Development

I'm a young developer, and this project has been a great experience for me in creating games. I tried to make the game as simple and fun as possible while staying within the strict size limits.
